# docs

Generated and handoff documentation produced by skill workflows.

## Responsibility

- Store durable chain outputs and decision artifacts used by downstream skills.
- Keep workflow facts discoverable across sessions.
- Preserve audit and planning outputs as local source-of-truth docs.

## Runtime Mapping

In a real project, this folder maps to:

`docs/ -> .agents/docs/`

## Typical Outputs

- `rb-audit-latest.json` capsules
- Idea and decision notes
- Planning and implementation handoff docs
- Workflow status or review summaries
